# 사칙연산 계산기
a + b = c 형태의 간단한 연산을 수행하는 콘솔 기반 계산기입니다.
- lv1 : 클래스를 사용하지 않고 구현
- lv2 : 클래스를 사용하여 구현
- lv3 : enum과 제네릭 응용

_____

### Lv1 주요 기능
- 정수/실수형의 사칙연산(+, -, /, *) 계산

### Lv2 주요 기능
- 정수/실수형의 사칙연산(+, -, /, *) 계산
- 계산식과 결과 저장
- 가장 오래된 결과 삭제

### Lv3 주요 기능
- 정수/실수형의 사칙연산(+, -, /, *) 계산
- 계산식과 결과 저장
- 가장 오래된 결과 삭제
- 지정 숫자보다 큰 결과로 필터링
